== Description ==

Cielos is a modern, fast, and accessible WordPress theme designed for blogs, portfolios, and business websites. Built with cutting-edge technologies including Vite, UnoCSS, Vue 3, and TypeScript, it offers excellent performance and developer experience.

**Key Features:**

* **Modern Build System**: Powered by Vite for lightning-fast development with Hot Module Replacement (HMR)
* **Atomic CSS**: Uses UnoCSS for highly optimized, on-demand CSS generation
* **Vue 3 Components**: Interactive components built with Vue 3 Composition API
* **TypeScript**: Full TypeScript support for better code quality
* **Dark Mode**: Built-in dark mode support with system preference detection
* **Accessibility Ready**: WCAG 2.1 compliant with skip links, ARIA attributes, and keyboard navigation
* **Block Editor Support**: Full Gutenberg compatibility with custom block styles
* **Responsive Design**: Mobile-first approach with fluid typography
* **Performance Optimized**: Lazy loading, code splitting, and optimized asset delivery

== Installation ==

1. In your WordPress admin panel, go to Appearance > Themes and click the "Add New" button.
2. Click "Upload Theme" and choose the downloaded zip file.
3. Click "Install Now" and then "Activate" to use the theme.

= Development Setup =

For developers who want to customize the theme:

1. Navigate to the theme directory
2. Run `pnpm install` to install dependencies
3. Run `pnpm dev` to start the development server with HMR
4. Run `pnpm build` to create production assets

== Frequently Asked Questions ==

= Does this theme support WooCommerce? =

Yes, Cielos includes basic WooCommerce support. You can use it for e-commerce websites.

= How do I enable dark mode? =

Dark mode is automatically enabled based on the user's system preferences. Users can also toggle between light, dark, and system modes using the theme toggle button in the header.

= What are the recommended plugins? =

* Breadcrumb NavXT - For breadcrumb navigation
* Advanced Custom Fields - For additional customization options
